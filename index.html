プライバシーポリシー

サービス名：AIニュース速報（以下「本サービス」）
制定日：2026年6月27日

1. はじめに

本サービスは、AIを活用して多言語（日本語・英語・中国語・韓国語・ベトナム語）のニュース動画を制作し、YouTube チャンネル「AIニュース速報」にて配信する個人運営のサービスです。本ポリシーは、本サービスの運営に用いるソフトウェア（以下「本アプリケーション」）における情報の取り扱いについて定めます。

2. 取得・アクセスする情報

本アプリケーションは、運営者自身の Google／YouTube アカウントに対して、Google OAuth 認証を通じてアクセスします。取得・アクセスする情報は次のとおりです。

  • 運営者自身の YouTube チャンネルに動画をアップロード・管理するために必要な認証情報（アクセストークン等）
  • 運営者自身がアップロードする動画およびそのメタデータ（タイトル、説明、再生リスト等）

本アプリケーションは、視聴者その他の第三者の個人情報を収集・取得しません。本アプリケーションを利用するのは運営者本人のみであり、一般の利用者に提供・配布していません。

3. 情報の利用目的

  • 運営者自身が制作した動画を、運営者自身の YouTube チャンネルへアップロードすること
  • 当該動画のタイトル・説明・再生リスト等を設定・管理すること

上記以外の目的で情報を利用することはありません。

4. 第三者への提供

本アプリケーションが API を通じて取得した情報を、第三者へ販売・提供・再配布・公開することは一切ありません。

5. Google API サービスの利用

本アプリケーションは YouTube API サービスおよび Google API を利用しています。これらの利用にあたり、本アプリケーションは以下に従います。

  • YouTube 利用規約
  • Google プライバシーポリシー

本アプリケーションによる Google ユーザーデータの取得・利用は、Google API Services User Data Policy（Limited Use の要件を含む）を遵守します。取得した Google ユーザーデータは、本ポリシー第3項に記載した機能の提供のためにのみ利用します。

6. 情報の保存と安全管理

OAuth 認証トークン等の認証情報は、運営者が管理する環境内に保存し、上記の目的以外には使用しません。不正アクセス・漏えい・滅失を防ぐため、適切な安全管理措置を講じます。不要となった認証情報は速やかに削除・無効化します。

7. コンテンツの開示について

本サービスの動画には合成音声および AI 生成コンテンツが含まれます。これらについては、YouTube の「変更されたコンテンツ・合成コンテンツ」の開示設定を有効にして配信します。

Privacy Policy (English)

Service: AIニュース速報 · Effective date: June 27, 2026

1. Overview

This service is an individually operated service that produces multilingual (Japanese, English, Chinese, Korean, Vietnamese) news videos using AI and publishes them on the YouTube channel "AIニュース速報". This policy describes how information is handled in the software (the "Application") used to operate this service.

2. Information We Access

The Application accesses the operator's own Google/YouTube account via Google OAuth, solely to obtain credentials needed to upload and manage videos on the operator's own YouTube channel, and to manage the operator's own uploaded videos and their metadata (title, description, playlists). The Application does not collect personal data of viewers or any third party. It is used only by the operator and is not distributed to the public.

3. How We Use Information

Information is used only to upload the operator's own videos to the operator's own YouTube channel and to set and manage their titles, descriptions, and playlists. It is not used for any other purpose.

4. Sharing of Information

Information obtained through the API is never sold, shared, redistributed, or publicly disclosed to any third party.

5. Use of Google API Services & Limited Use

The Application uses YouTube API Services and Google APIs in accordance with the YouTube Terms of Service and the Google Privacy Policy. The Application's use of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

6. Storage & Security

Credentials such as OAuth tokens are stored within an environment controlled by the operator and are used only for the purposes above. Appropriate safeguards are applied, and credentials are promptly deleted or revoked when no longer needed.
